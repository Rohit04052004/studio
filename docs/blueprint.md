# **App Name**: MedReport Clarity

## Core Features:

- Report Upload: Allow users to upload medical reports in text or image format (chest X-rays, MRIs etc.).
- AI Summary Generation: Use Gemini API and Langchain to automatically generate simple summaries of the lab reports and scans and explain medical terms.
- Abnormal Result Highlighting: Emphasize any abnormal or significant findings in the generated summary. The LLM will act as a tool, and may or may not use it's highlighting feature.
- Chat Interface: Enable users to ask questions about their reports and health topics.
- AI-Powered Answers: Provide evidence-based answers using RAG (Retrieval-Augmented Generation) powered by the Gemini API.
- Report Display: Display uploaded reports and generated summaries in a clear and user-friendly format.
- User Authentication: Implement user authentication with secure registration/login.
